# 英語格鬥｜113年會考英語閱讀挑戰

以原創街機格鬥介面呈現113年（2024）國中教育會考英語閱讀公開試題；本專案與114年版本完全獨立。

## 內容

- 英語閱讀43題完整題庫；每場隨機12題
- 43張單題圖、9張共用文章／情境圖
- 三位角色、HP、連擊、能量、大招與每題75秒倒數
- 答題後顯示正答、具體解析與常見誤區
- 結算、歷史最佳與錯題重練（只存本機瀏覽器）
- 手機題圖卡內橫向滑讀、全螢幕燈箱、鍵盤A–D作答
- 音效、減少動態、高對比

## 年份入口

- 年份選擇：<https://addielu-phy.github.io/cap-english-fighter-years/>
- 114年版本：<https://addielu-phy.github.io/cap-english-fighter/>

113與114使用不同repo及localStorage命名空間，題庫、最佳成績與錯題不會互相覆蓋。

## 本機執行

```sh
python -m http.server 4174 --bind 127.0.0.1
```

開啟 <http://127.0.0.1:4174/>。

## 重建題庫

`data/exam113.json`凍結官方答案與原始題圖關係；建置器會按檔名將來源路徑明確映射到公開的`assets/questions/`。`data/meta113.json`保存逐題教學解析：

```sh
npm run build:data
```

## 測試

```sh
npm install
npm test
```
